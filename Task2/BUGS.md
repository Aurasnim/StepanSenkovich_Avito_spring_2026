1) BUG-UI-001: Некорректная работа фильтра "Диапазон цен".
   Приоритет: P1.
   Серьезность: Major.
   Окружение: Windows-система/веб-версия сайта "Модерация Авито"
   Шаги воспроизведения:
         1. Открыть страницу "Объявления" сайта "Модерация Авито".
         2. Ввести значение «50000» в поле «От» фильтра «Диапазон цен».
         3. Ввести значение «60000» в поле «До» фильтра «Диапазон цен».
   Ожидаемый результат: В выдаче отображаются только объявления, цена которых находится в диапазоне от 50000 до 60000.
   Фактический результат: В выдаче присутствуют объявления, выходящие за рамки диапазона.
   Дата обнаружения: 02.04.2026
   Автор: Aurasnim

2) BUG-UI-002: Отсутствие активации кнопки автообновления.
   Приоритет: P0.
   Серьезность: Critical.
   Окружение: Windows-система/веб-версия сайта "Модерация Авито"
   Шаги воспроизведения:
         1. Открыть страницу "Статистика" сайта "Модерация Авито".
         2. Нажать на кнопку отключения автообновления.
         3. Нажать на кнопку повторного запуска автообновления.
   Ожидаемый результат: Автообновление повторно запускается.
   Фактический результат: Кнопка повторного запуска автообновления не активна.
   Дата обнаружения: 02.04.2026
   Автор: Aurasnim

3) BUG-UI-003: Некорректная работа тоггла "Только срочные"
   Приоритет: P1.
   Серьезность: Major.
   Окружение: Windows-система/веб-версия сайта "Модерация Авито"
   Шаги воспроизведения:
         1. Открыть страницу "Объявления" сайта "Модерация Авито".
         2. Активировать тоггл "Только срочные"

   Фактический результат: Помимо релевантных объявлений в выдаче отображаются оъявления без тега "Срочно".
