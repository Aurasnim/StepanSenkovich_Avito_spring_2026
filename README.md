# StepanSenkovich_Avito_spring_2026
Репозиторий тестового задания на стажировку в Авито.

Автоматизированные тесты для платформы модерации Avito

Данный проект содержит автоматизированные тесты для платформы модерации объявлений Avito, разработанные с использованием pytest и Playwright.

Требования:
Python 3.8+
pip (менеджер пакетов Python)

1) Установка
    1. Клонируйте репозиторий (если применимо):
    git clone <URL_репозитория>
    cd <имя_репозитория>
    
    2. Установите зависимости Python:
    pip install pytest pytest-playwright
    
    
    3. Установите браузеры, необходимые для Playwright (Chromium):
    playwright install chromium

2) Запуск тестов
    Для запуска всех тестов используйте следующую команду:
    pytest test_avito_moderation.py --browser chromium
    
    test_avito_moderation.py: Имя файла с тестами.
    --browser chromium: Указывает Playwright использовать браузер Chromium.
    Запуск тестов в режиме без графического интерфейса (headless)
    По умолчанию Playwright запускает браузер в режиме headless. Если вы хотите увидеть работу браузера, уберите флаг --headless.

3) Отчет о выполнении
    Тесты настроены таким образом, что при обнаружении бага они будут падать, явно указывая на проблему. В выводе pytest будет отображена информация о проваленных тестах и причинах их падения.
    Пример вывода при обнаружении бага:
    Plain Text
    
    =================================== FAILURES ===================================
    _____________________ test_price_filter_invalid[chromium] ______________________
    ...
    E       Failed: BUG P3: Поле 'От' приняло отрицательное значение
    ...
    ========================= 1 failed, 5 passed in X.XXs ==========================
    
    
    Это позволяет быстро идентифицировать проблемные места и их приоритет (P1, P2, P3), указанный в сообщении о падении теста.
